# Status View Specification

> **Version**: 1.0  
> **Last Updated**: 2026-01-11

## 개요

채팅 화면 헤더의 Partner 아이콘을 탭하면 나타나는 상태 정보 모달.
Partner, User, Director 정보를 탭으로 전환하며 확인 가능.

## 탭 구성

### 1. Partner 탭

#### A. 프로필 이미지
- 초기 생성된 Partner 이미지
- 탭하면 전체 화면

#### B. 기본 정보
```
사쿠라, 22세
대학생
```

#### C. 감정 상태 (실시간)
```
💖 감정 상태
• Affection: ████████░░ 65/100
• Trust: ████████░░ 70/100
• Arousal: ██████░░░░ 60/100
• Lust: ████░░░░░░ 45/100
• Dominance: ░░░░░█░░░░ -20 (-100 to +100)
• Mood: Flustered 😳
```

**프로그레스 바**:
- 0-100 값에 따라 채워짐
- 색상:
  - Affection: 핑크
  - Trust: 파란색
  - Arousal: 주황색
  - Lust: 빨간색
  - Dominance: 보라색 (중앙 기준)

**Mood 아이콘**:
- Happy: 😊
- Flustered: 😳
- Anxious: 😰
- Horny: 😍
- Angry: 😠
- Sad: 😢

#### D. 성격
```
👤 성격
• MBTI: INFP
• 말투: 존댓말
• 특성: 내향적, 감성적, 배려심 많음
```

#### E. 외모
```
🎨 외모
• 얼굴: 큰 눈과 부드러운 미소
• 헤어: 금발 포니테일
• 체형: 날씬하고 우아한
• 의상: 흰색 블라우스, 검은색 치마
```

**NSFW 모드 시**:
- 의상 → NSFW Outfit으로 변경

#### F. 비밀
```
🔒 비밀 (Trust 80+ 시 공개)
• ???
```

**Trust < 80**:
- 잠금 아이콘
- "???" 표시

**Trust >= 80**:
```
🔓 비밀
• 실은 유명 작가의 딸이지만 숨기고 있다
```

#### G. 트라우마 (감지된 경우만)
```
💔 트라우마
• 아버지가 유명 작가라는 사실이 밝혀짐
```

### 3. Director 탭

#### A. 세계 상태
```

• 시간: 2024-01-10 18:30 (저녁)
• 날씨: rainy 🌧️
• 위치:
  - Partner: park
  - User: park
```

#### B. 현재 상황
```
📍 현재 상황
"비가 내리고 옷이 젖어가는 중"
```

#### C. 진행 중인 이벤트
```
🎬 진행 중인 이벤트
• 없음
```

**이벤트 있을 경우**:
```
🎬 진행 중인 이벤트
• 벚꽃 축제
```

#### D. 턴 정보
```
📊 진행 상황
• 턴 수: 15
• 장르: Pure Romance
```

#### E. 의상 변경 (있을 경우)
```
👔 의상 변경
• Partner: 흰색 블라우스 → 검은색 란제리
• User: 변경 없음
```

## 상호작용

### 탭 전환
- 탭 버튼 클릭/탭으로 전환
- 부드러운 페이드 애니메이션

### 닫기
- 우측 상단 ✕ 버튼
- 모달 외부 탭
- 뒤로 가기 제스처

### 이미지 확대
- Partner/User 이미지 탭 시 전체 화면

## 애니메이션

### 모달 등장
- 하단에서 슬라이드 업
- 배경 어둡게 (반투명)

### 탭 전환
- 페이드 인/아웃
- 슬라이드 효과 (선택적)

### 프로그레스 바
- 값 변경 시 부드러운 애니메이션
